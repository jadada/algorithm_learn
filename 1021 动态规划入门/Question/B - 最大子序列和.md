# B : 最大子序列和

Time Limit: `1 Sec`,   Memory Limit: `128 Mb`

## Description

给定一个整数数组，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

## Input

有多组测试数据。 对于每组测试数据，第一行只有一个整数N，代表着数组的大小(1 <= N <= 100000)

第二行有N个整数，(-100 <= N <= 100)

## Output

每组测试数据仅输出一行，包括一个整数，表示最大子序列和。

## Sample Input

```
9
-2 1 -3 4 -1 2 1 -5 4
```

## Sample Output

```
6
```
