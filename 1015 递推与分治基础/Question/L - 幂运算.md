# L : 幂运算

Time Limit: `1 Sec`,   Memory Limit: `128 Mb`

## Description

给你三个整数a,b,q，求 $ab$ mod $p$ 的值

## Input

第一行是一个整数*t*，表示*t*组数据。

接下来的n行，每行有 *3* 个整数，分别表示*a*, *b*, *p*

$t ≤ 2 \times 10^5$，$a > 0, b > 0, p ≥ 2$,

## Output

每组数据，输出一个整数表示答案。

## Sample Input

```
2
2 10 9
2 3 3
```

## Sample Output

```
7
2
```
