# B : 如何溜的最快

Time Limit: `1 Sec`, Memory Limit: `128 Mb`

## Description

给你一个终点 $(x,y)$，从 $(0,0)$ 出发，一步最长为 $R$，可以走向任意方向，请你输出到达终点所需的最短步数。

## Input

第一行输入 $r,x,y$

$1 \leq R \leq 10^5$

$0 \leq x,y \leq 10^5$

$(x,y) \neq (0,0)$

所有输入皆为int型

## Output

输出结果并换行

## Sample Input

```
3 4 4
```

## Sample Output

```
2
```
