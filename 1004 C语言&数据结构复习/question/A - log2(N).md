# A : log2(N)

Time Limit: `1 Sec`, Memory Limit: `128 Mb`

## Description

给你一个数 $N$，请你找出最大的的整数 $k$，满足 $2^k \leq N$

## Input

$1 \leq N \leq 10^{19}$

## Output

输出 $k$ 并换行

## Sample Input

```
6
```

## Sample Output

```
2
```
