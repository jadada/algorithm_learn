# C : 200和整数对之间的情缘

Time Limit: `1 Sec`,   Memory Limit: `128 Mb`

## Description

给你一个有 $N$ 个整型数字的序列 $A$，整数对 $(i,j)$ 满足以下条件，请问能找到多少这样的整数对：

- $1 \leq i,j \leq N$
- $A_i-A_j$ 是200的整数倍

## Input

第一行输入 $N$，第二行输入 $N$ 个整数分别 $A_1,A_2,\cdots,A_N$

- 所有输入均为整型(int)
- $2 \leq N \leq 2\times10^5$
- $1 \leq A_i \leq 10^9$

## Output

输出答案并换行

## Sample Input

```
6
123 223 123 523 200 2000
```

## Sample Output

```
4
```
