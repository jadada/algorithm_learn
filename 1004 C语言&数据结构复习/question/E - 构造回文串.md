# E : 构造回文串

Time Limit: `1 Sec`, Memory Limit: `128 Mb`

## Description

给你一个整型(int)数 $N$，可以将其看成字符串，是否有可能在 $N$ 前加入若干个0(也可以不加)，使得该字符串为回文串。

## Input

数字 $N$ ($0 \leq N \leq 10^9$)

## Output

若可以则输出Yes，不可以则输出No，输出结束要换行。

## Sample Input

```
1210
```

## Sample Output

```
Yes
```
