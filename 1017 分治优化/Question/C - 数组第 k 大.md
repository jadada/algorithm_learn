# C : 数组第 k 大

Time Limit: `1 Sec`,   Memory Limit: `128 Mb`

## Description

给出 n 个数，以及 k ，求数组中的第 k 大的数。

## Input

一个测试文件内有多组数据，每组数据 *n* + 1 行，第一行正整数 n, k，接下来 n 个正整数 $a_i$

其中， $1 ≤ k ≤ n ≤ 10^5, 1 ≤ a_i ≤ 10^9$

## Output

对每组数据输出第 k 大的数

## Sample Input

```
5 2
1 3 5 7 9
```

## Sample Output

```
7
```
