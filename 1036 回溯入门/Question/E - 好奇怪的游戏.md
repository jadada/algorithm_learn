# E : 好奇怪的游戏

## Description

这个游戏类似象棋，但是只有黑白马各一匹，在点 *x*1, *y*1 和 *x*2, *y*2 上。它们得从点 *x*1, *y*1 和 *x*2, *y*2 走到 (1,1) 。这个游戏与普通象棋不同的地方是：马可以走“日”，也可以像象走“田”。现在想知道两匹马到 (1,1) 的最少步数，你能解决这个问题么？

## Input

第1行：两个整数*x*1, *y*1

第2行：两个整数*x*2, *y*2

(1≤*x*1,*y*1,*x*2,*y*2≤20)

## Output

第1行：黑马到(1,1)的步数

第2行：白马到(1,1)的步数

假设棋盘为`20*20`

## Sample Input

```
12 16
18 10
```

## Sample Output

```
8 
9
```
